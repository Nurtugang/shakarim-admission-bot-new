Shakarim Admission Bot API

Добро пожаловать на API страницу бота-помощника для поступающих в НАО "Шәкәрім университет". Этот проект разработан в рамках инновационной программы AI Sana.

Доступные API эндпоинты:

Умный ответ с учетом базы знаний

URL: /api/smart_ask_gemini/

Метод: GET

Параметры: question - вопрос пользователя

Пример: /api/smart_ask_gemini/?question=Какие есть гранты?

Управление базой знаний

URL: /admin/api/knowledgebase/

Назначение: доступ к редактированию базы знаний через Django Admin

Требуется авторизация.

Пример: Перейти в админку
