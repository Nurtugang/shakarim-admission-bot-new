Shakarim Admission Bot API

Добро пожаловать на API страницу бота-помощника для поступающих в НАО "Шәкәрім университет" Семей. Этот проект разработан в рамках инновационной программы AI Sana.

Доступные API эндпоинты:

Получить ответ от Gemini

URL: /api/ask_gemini/

Метод: GET

Параметры: question - вопрос пользователя

Пример: /api/ask_gemini/?question=Какие документы нужны для поступления?

Умный ответ с учетом базы знаний

URL: /api/smart_ask_gemini/

Метод: GET

Параметры: question - вопрос пользователя

Пример: /api/smart_ask_gemini/?question=Какие есть гранты?

Получить знания из базы

URL: /api/get_knowledge/

Метод: GET

Параметры: category (опционально) - категория знаний

Пример: /api/get_knowledge/?category=гранты

Добавить знание в базу

URL: /api/add_knowledge/

Метод: POST

Тело запроса:

{
  "category": "документы",
  "text": "Информация о документах для поступления."
}

Удалить знание из базы

URL: /api/delete_knowledge/<doc_id>/

Метод: DELETE

Пример: /api/delete_knowledge/fUuYi6ojkJW...
